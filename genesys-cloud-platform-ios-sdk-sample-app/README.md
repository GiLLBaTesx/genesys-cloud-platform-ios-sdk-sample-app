# agent-monitoring-ios-app

## Project setup

```bash
pod install
```

## Open project in Xcode

```bash
open GenesysCloudConnectDemo.xcworkspace
```

## Build and run for development

Press `⌘+R` in Xcode or:

```bash
xcodebuild -workspace GenesysCloudConnectDemo.xcworkspace -scheme GenesysCloudConnectDemo -configuration Debug
```

## Build for production

```bash
xcodebuild -workspace GenesysCloudConnectDemo.xcworkspace -scheme GenesysCloudConnectDemo -configuration Release
```

## Clean build folder

Press `⌘+Shift+K` in Xcode or:

```bash
xcodebuild clean -workspace GenesysCloudConnectDemo.xcworkspace -scheme GenesysCloudConnectDemo
```

## Update dependencies

```bash
pod update
```

See [Xcode Build Settings Reference](https://developer.apple.com/documentation/xcode/build-settings-reference)
